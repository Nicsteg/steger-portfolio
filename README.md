# 3D Portfolio

An interactive 3D developer portfolio built with React, Vite, and React Three Fiber.

## Overview
This portfolio presents your background and projects through a stylized 3D island scene with animated elements and navigable portfolio sections.

## Tech stack
- React
- Vite
- React Three Fiber
- Drei
- React Router
- EmailJS

## Features
- interactive 3D landing page
- draggable/rotating island scene
- staged intro panels
- animated models and scene elements
- contact workflow via EmailJS
- portfolio pages for about, projects, and contact
- optional background music toggle

## Getting started
```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.

## Production
```bash
npm run build
npm run preview
```

## Useful scripts
- `npm run dev` - start local dev server
- `npm run build` - create production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint

## Project structure
- `src/pages/` - app pages
- `src/components/` - reusable UI pieces
- `src/models/` - 3D scene models
- `src/constants/` - content/config
- `src/hooks/` - custom hooks
- `src/assets/` - icons, audio, and static assets

## Notes
- The built `dist/` folder is currently committed.
- Main landing experience is driven from `src/pages/Home.jsx`.
- Home messaging cards are configured in `src/components/HomeInfo.jsx`.
